# Finance Tracker and Bill Manager

Finance Tracker and Bill Manager is a comprehensive web application designed to help individuals track their finances, manage bills, and monitor their progress towards financial goals. This single-page application offers a seamless user experience with insightful analytics and easy management of financial data, leveraging technologies like HTML, CSS, JavaScript, Node.js, MySQL, and Express.

## Features

- **Single Page Application:** Efficient navigation without reloading pages.
- **Expense and Income Tracking:** Categorize and monitor money inflow and outflow.
- **Bill Management:** Automated reminders and updates for bill payments.
- **Financial Goals Monitoring:** Set and track financial targets.
- **Dynamic Dashboard:** Visual representation of income vs. expense ratio, bill payment status, and net earnings over different periods.
- **Customizable UI:** Night mode preference for enhanced user experience.
- **User Profile Management:** Edit personal credentials and customize settings.

## Built With

- HTML, CSS, JavaScript – Frontend
- Node.js, Express – Backend
- MySQL (phpMyAdmin) – Database
- Express-Session – User Session Management
- Fetch API – Asynchronous requests
